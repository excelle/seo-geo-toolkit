# /seo:report

Generates a stakeholder-ready performance report.

## Usage

```
/seo:report <domain> [period]
```

## What It Does

1. Invokes [signal-reporter](../track/signal-reporter/SKILL.md)
2. Compiles traffic, ranking, backlink, and AI citation metrics
3. Produces narrative report with recommendations

## Output

- Executive summary
- KPI dashboard
- Wins and opportunities
- 3 prioritized next actions

## Example

```
/seo:report example.com "Q1 2026"
```
