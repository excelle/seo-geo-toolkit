# Contributing

Thanks for helping improve the OpenCode SEO & GEO Toolkit.

## How to Contribute

1. **Fork** the repository
2. **Create a branch**: `feature/your-engine-name` or `fix/description`
3. **Write your engine** following the format in existing `SKILL.md` files
4. **Add references** if detail exceeds 350 lines
5. **Update tracking files**: `VERSIONS.md`, `README.md`, `OPENCODE.md`
6. **Open a pull request** with a clear description

## Engine Requirements

- Frontmatter with `name`, `version`, `description`, `license`, `compatibility`, `metadata`
- Clear trigger phrases in `metadata.triggers`
- Quick Start section with example prompts
- Engine Contract section (reads, writes, promotes, next handoff)
- Instructions with numbered steps
- Next Best Engine recommendation
- Under 350 lines (detail in `references/`)

## Style Guide

- Second person, direct tone
- Bold key terms on first use
- Tables for structured data; code blocks for commands
- Be specific — every recommendation should include a number or concrete example

## Code of Conduct

Be respectful, constructive, and inclusive.
